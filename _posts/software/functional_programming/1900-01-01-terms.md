## Option == Maybe
null 사용으로 인한 여러 문제들을 회피하기 위해 사용하는 기법. Functional Programming에서는 Monad로 만들어 사용한다.

```
Option := Some | None
Maybe := Just | Nothing
```

JavaScript와 Haskell에서는 Maybe라는 용어를 사용한다.

C++에서는 _std::optional_ 을 C++17 부터 지원한다. 그리고 C++23부터 아래와 같은 monadic operation을 지원한다.
* [and_then](https://en.cppreference.com/w/cpp/utility/optional/and_then)
* [transform](https://en.cppreference.com/w/cpp/utility/optional/transform)
* [or_else](https://en.cppreference.com/w/cpp/utility/optional/or_else)
