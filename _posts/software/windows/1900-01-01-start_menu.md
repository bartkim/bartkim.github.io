시작메뉴에 특정 프로그램 등록하기 위해서는 아래 경로에 원하는 프로그램의 바로가기를 만들어 넣으면 된다.


## 특정 사용자
```
%AppData%\Microsoft\Windows\Start Menu\Programs
```

## 모든 사용자
```
C:\ProgramData\Microsoft\Windows\Start Menu\Programs
```
